I acted as a Machine Learning Developer, focusing on building an intelligent system to automate the detection of spam communications.
The project involved implementing Natural Language Processing (NLP) techniques to bridge the gap between human language and machine logic.
I utilized TF-IDF Vectorization to extract key features, converting raw email text into numerical representations for the models to process.
I took a rigorous approach to model selection by training multiple algorithms, including Multinomial Naive Bayes, Logistic Regression, and SVM.
Each model was carefully evaluated based on its ability to minimize false positives while maintaining a high overall detection rate.
I managed the entire development pipeline, from initial data preprocessing and text cleaning to final model performance testing.
This project showcased my ability to handle high-dimensional text data and produce a functional binary classification tool.
My contribution ensured the creation of a reliable filter that can effectively protect users by identifying and isolating malicious spam emails.
